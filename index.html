Què és Flexbox?

Flexbox (Flexible Box Layout) és un model de disseny de CSS3 per distribuir espai i alinear elements dins d’un contenidor de forma eficient i flexible.

Configuració bàsica


<div class="contenidor">
  <div class="element">1</div>
  <div class="element">2</div>
  <div class="element">3</div>
</div>
.contenidor {
  display: flex;
}

Propietats del contenidor

1. display

display: flex; estableix el contenidor com un flex container, activant totes les funcionalitats de Flexbox. display: inline-flex; fa el mateix però com a element inline.

2. flex-direction

Defineix la direcció principal en què es col·loquen els elements fills dins el contenidor.

  • row : en fila, de esquerra a dreta (per defecte)
  • row-reverse : en fila, de dreta a esquerra
  • column : en columna, de dalt a baix
  • column-reverse : en columna, de baix a dalt

3. justify-content

Controla la distribució dels elements al llarg de l’eix principal (definit per flex-direction ).

  • flex-start : elements alineats a l'inici
  • flex-end : elements alineats al final
  • center : centrats
  • space-between : espai igual entre ells
  • space-around : espai als costats i entre ells
  • space-evenly : espai totalment uniforme

4. align-items

Defineix l’alineació dels ítems a l’eix perpendicular a l’eix principal.

  • stretch : s’estiren per omplir l’espai (per defecte)
  • flex-start : alineats a l’inici
  • flex-end : alineats al final
  • center : centrats
  • baseline : alineats segons la línia de base del text

6. align-content

S’utilitza quan hi ha múltiples files (amb flex-wrap). Controla l’alineació de les files dins del contenidor.

7. flex-wrap

Permet que els elements es moguin a la següent línia si no hi ha prou espai.

  • nowrap : tot en una sola línia (per defecte)
  • wrap : els ítems poden saltar a la línia següent
  • wrap-reverse : igual que wrap però invertint l’ordre de les línies

8. gap

Defineix l'espai entre ítems del contenidor té dues propietats (row-gap i column-gap)

  • row-gap: Afecta a l'espai que hi ha entre files espai en horitzontal
  • column-gap: Afecta a l'espai que hi ha entre columnes espai en vertical
  • gap: O en Aquest Cas adjunta les dues propietats el row-gap i el column-gap si poses un valor els dues tenen el mateix valor si posem dos valors cada propietat té un valor gap:row-gap column-gap;

Propietats dels elements (fills)

1. order

Canvia l'ordre visual dels elements dins del contenidor sense alterar l'ordre HTML.

2. flex-grow

Defineix la proporció amb què un element pot créixer si hi ha espai disponible.

3. flex-shrink

Controla com un element pot reduir la seva mida si no hi ha prou espai.

4. flex-basis

Defineix la mida base d’un element abans que s’apliquin les regles de creixement o reducció.

6. align-self

Sobreescriu align-items per un ítem concret.
